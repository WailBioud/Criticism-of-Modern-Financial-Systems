# Central Bank System Simulation

A secure client-server banking system simulation designed for Kali Linux.

## Features
- Secure encrypted communication
- Transaction monitoring
- Client authentication
- Money transfer with limits
- Transaction signing

## Architecture
- Server: Central authority managing accounts and transactions
- Client: User interface for banking operations

## Requirements
- Python 3.8+
- PyCryptodome library
